Custom AT commands

Commands run exactly as typed, one per line (max 20). Write commands (CFUN, QCFG, CRSM…) are allowed — use with care.

Command reference

Click a command to add it to the box above.

Forbidden network list (FPLMN)

AT+CRSM=176,28539,0,0,12 Read the SIM's blacklist of rejected networks. Reply is groups of 6 hex chars, one network each; FFFFFF = empty slot. A non-empty list can block the modem from carriers your plan supports — it persists across reboots.
AT+CRSM=214,… + AT+CFUN=1,1 Clear the blacklist, then reboot the modem so it rescans all carriers with a clean slate. (The connection drops for ~1 min while the modem restarts.)

Network selection

AT+COPS? Current operator and selection mode (0 auto / 1 manual).
AT+COPS=? Scan all visible networks — takes 1–2 minutes, run it alone. Leading digit per entry: 1 available · 2 current · 3 forbidden. May answer ERROR if the modem is busy registering; retry.
AT+COPS=1,2,"310260" Force a specific network by PLMN code (US: 310260 T-Mobile · 310410 AT&T · 311480 Verizon). Persists across reboots — return to auto when done testing.
AT+COPS=0 Back to automatic network selection.

Quick status

AT+CSQ Signal: first number 0–31 (higher = better; 99 = no signal / not camped). dBm ≈ −113 + 2×n.
AT+CEREG? LTE registration: second digit 1 home · 5 roaming (normal for Hologram) · 2 searching · 3 denied.
AT+QNWINFO Serving network, PLMN, and band (Quectel only).
